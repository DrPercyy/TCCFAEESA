Qual é a principal característica da linguagem Python?
A) Linguagem de programação compilada.
B) Linguagem de programação orientada a objetos.
C) Linguagem de programação fortemente tipada.
D) Linguagem de programação de baixo nível.
E) Linguagem de programação baseada em lógica fuzzy.
Resposta: B

O que é uma lista em Python?
A) Um tipo de variável que só pode conter números inteiros.
B) Um tipo de estrutura de controle que repete um bloco de código.
C) Um tipo de dado que armazena valores em pares chave-valor.
