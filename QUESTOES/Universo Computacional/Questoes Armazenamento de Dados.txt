O que é um disco rígido (HD) em um computador?
A) Uma unidade de processamento central.
B) Um dispositivo de entrada de dados.
C) Uma unidade de armazenamento de dados não volátil que utiliza discos magnéticos para armazenar informações.
D) Um dispositivo de saída de dados.
E) Uma memória RAM.
Resposta: C
